Esse projeto tem como objetivo traduzir as texturas e os textos de Ricochet para outros idiomas que não possuem uma tradução oficial, de maneira comunitária, de fã para fã.

Essas traduções são feitas pela comunidade, este repositório agrega as texturas e textos da comunidade para o jogo.

> [!WARNING]
> O repositório com a interface se encontra privado no momento.

## Traduções disponíveis
| Idioma | Textos | Texturas | 
| ------ | ------ | -------- |
| Português | ✅ 100% | ✅ 100% |

- [Texturas traduzidas](../../Docs/PT-BR/Texturas.md)

## Como colaborarar 
Este repositório contem as texturas de maneira editável, se você possue conhecimento, você pode trazer as texturas para o seu idioma, ou melhorar as texturas já existentes para ficar mais proxímo a textura original.

1. **Sugestões de melhorias ou correções**

   * Abra uma *issue* neste repositório
   * Ou entre em contato via Discord

2. **Traduções para outros idiomas**

   * Entre em contato com a equipe para orientações
   * Após validação, sua tradução será adicionada ao projeto com os devidos créditos

> [!NOTE]
> Este projeto é gerenciado pela Source BR, então caso tenha alguma dúvida entre no nosso [Discord](https://discord.gg/tVNv6SNZZT)
